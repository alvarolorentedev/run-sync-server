# Motivation

Server for run-sync.com

# Requirements

* Node.js >= 7.5.x
* yarn

# Usage

to install this project 

```bash
git clone https://github.com/kanekotic/run-sync-server.git
yarn install 
#start commands
yarn start
yarn start:dev
yarn start:prod
#test commands
yarn test
yarn test:dev
yarn test:cov
```
